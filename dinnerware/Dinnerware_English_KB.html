Dinnerware Knowledge Base

FACTS ABOUT CUTLERY A knife can't be used for soup A fork can't be used for soup A spoon can be used for soup Chopsticks can be used for soup FACTS ABOUT DISHES A plate can't be used for soup A bowl can be used for soup A cup can be used for soup A saucecup can't be used for soup RULES FOR LISTING CUTLERY AND DISHES AND DINNERWARE If you sequentially, exhaustively, grab a cutlery and write its name then you will have written the names of all the cutlery. If you sequentially, exhaustively, grab a dish and write its name then you will have written the names of all the dish. If you write the names of all the cutlery and then write the names of all the dishes then you will have written the names of all the shapes. RULES FOR FINDING THINGS TO EAT SOUP WITH If a cutlery can soup then you can use it to eat soup. If a dishes can soup then you can use it to eat soup. If a cutlery can be used to eat soup and a dish can be used to eat soup then they can be used together to eat soup. If a cutlery cannot be used to eat soup and a dish cannot be used to eat soup then they cannot be used together to eat soup. UTILITIES If you type exit then the program should stop running like every other command line program.
